■ 이 확장이 하는 일

네이버 뉴스의 "내 댓글 모음" 창에 표시되는 본인의 댓글을 자동으로 순차 삭제합니다.
그동안 남긴 오래된 댓글을 한 건씩 수동으로 지우기 지쳤을 때 유용합니다.

■ 사용 방법

1. 네이버 뉴스 기사 URL 뒤에 #user_comment 를 붙여 접속합니다.
   예: https://n.news.naver.com/article/000/0000000000#user_comment
2. 우측에 "뉴스 댓글모음" 창이 열리면 툴바의 확장 아이콘을 클릭합니다.
3. 팝업의 [시작] 버튼을 누르면 삭제 버튼을 반복 클릭합니다.
4. 언제든지 [중지] 버튼으로 중단할 수 있습니다.
5. 삭제할 댓글이 더 이상 없으면 자동으로 종료됩니다.

■ 안전에 관한 안내

- 본 확장은 본인 계정으로 로그인한 상태에서, 본인이 작성한 댓글만 삭제합니다.
- 네이버 서비스는 본인 댓글의 삭제 권한만 부여하므로 타인의 댓글은 물리적으로 삭제할 수 없습니다.
- 삭제한 댓글은 되돌릴 수 없으니 신중히 사용하세요.
- 이 확장은 어떠한 개인정보도 수집하거나 외부로 전송하지 않습니다.

■ 개인정보 처리

- 데이터 수집 없음
- 외부 서버 통신 없음
- 로컬 저장소 사용 없음
- 광고/분석 도구 없음

자세한 개인정보처리방침은 아래 링크를 참고하세요.

■ 요청 권한 안내

## 8. 권한 사용 사유 (Permission Justifications)

각 권한별로 별도 입력란이 있습니다.

### 8-1. scripting

```
사용자가 팝업의 '시작' 버튼을 클릭했을 때, 현재 열린 네이버 뉴스 탭에 chrome.scripting.executeScript로 자동 삭제 함수를 주입하기 위해 필요합니다. 페이지의 window.confirm을 임시로 오버라이드하여 삭제 확인 다이얼로그를 자동 승인하는 데에도 사용됩니다.
```

### 8-2. activeTab

```
현재 활성 탭이 네이버 도메인(*.naver.com)인지 확인하고, 그 탭에만 스크립트를 주입하기 위해 필요합니다. activeTab은 사용자가 확장 아이콘을 클릭한 순간에만 권한이 부여되므로 프라이버시에 안전합니다.
```

### 8-3. host_permissions: `*://*.naver.com/*`

```
네이버 뉴스, 네이버 로그인 도메인 등 네이버 계열 페이지 전반에서 '내 댓글' 창의 삭제 버튼을 인식하고 클릭하기 위해 필요합니다. 이 권한은 네이버 도메인에만 국한되며, 다른 웹사이트에서는 확장이 전혀 동작하지 않습니다.
```
